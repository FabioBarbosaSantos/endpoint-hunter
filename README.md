# endpoint-hunter

JavaScript Endpoint Extractor written in Python.

Installation

Clone the repository:

```bash
git clone https://github.com/FabioBarbosaSantos/endpoint-hunter.git
cd endpoint-hunter
pip install -r requirements.txt
```

Run the tool:

```bash
python3 main.py https://example.com
```

Verbouse mode:
```bash
python3 main.py https://example.com -v
```
